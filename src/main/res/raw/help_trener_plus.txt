1 Назначение приложения.

Программа предназначена для точного дозирования циклической нагрузки за счёт задания темпа движений в процессе её выполнения. Особенно удобно применение темполидера в упражнениях с выраженными моментами силовых напряжений – подтягиваниях, отжиманиях, приседаниях и т.п. 

Темполидер позволяет запоминать время выполнения каждого цикла движений спортсмена и затем воспроизводить его с помощью звуковых сигналов, что позволяет добиться идентичности нагрузки в двух и более подходах. Кроме того, предусмотрена возможность оперативного изменения темпа (как увеличение, так и снижение) на любую заданную в процентах величину.

Если темп в ходе выполнения упражнения изменяется скачкообразно – как это часто бывает в подтягивании на перекладине, когда приходится согласовывать циклы подтягиваний с циклами дыхания – время выполнения цикла движений может быть одинаковым для нескольких циклов подряд. В этом случае подход фактически распадается  на фрагменты с различным темпом движений, что позволяет быстро и удобно вручную составлять раскладки по темпу, задавая время выполнения цикла и количество повторений для всех фрагментов подхода

Темполидер,  позволяя воспроизводить  раскладку по темпу любой степени сложности, даёт возможность контролировать темп выполнения упражнения от подхода к подходу, от тренировки к тренировке, помогая объективно оценивать изменение уровня работоспособности спортсмена.


2 Экран «Главное меню». Выбор варианта действий.

Пункт «Секундомер» открывает экран «Секундомер» с отсечками и возможностью сохранения отсечек в виде раскладки секундомера, где количество повторений каждого фрагмента равно 1.

Сохранённые отсечки можно просмотреть на графике и визуально оценить степень изменения темпа в процессе выполнения упражнения.

Пункт «Темполидер» открывает экран «Темполидер» который помогает выполнить циклическое упражнение в соответствии с заданной раскладкой по темпу со звуковым сопровождением.

Пункт «Раскладки» открывает экран «Раскладки» с тремя вкладками, содержащими списки раскладок, сохранённых в базе данных приложения.


2.1 Создание новой раскладки по темпу.

2.1.1  Через круглую кнопку   «+»  в главном меню

Введите данные хотя бы для одного фрагмента подхода, укажите время задержки старта, дайте раскладке имя, а затем нажмите «Создать раскладку»

2.1.2  На основе ранее созданной раскладки

Для этого выберите пункт «Раскладки» в главном меню, а затем выберите подходящую раскладку на одной из трёх вкладок. Выбранная раскладка откроется на экране темполидера. Для редактирования раскладки перейдите в редактор раскладок, нажав на иконку карандаша на панели инструментов в верхней части экрана.

 Доступно изменение параметров для всех фрагментов сразу или для любого выбранного фрагмента в зависимости от установки пункта «Выполнить для всех строк», Добавление, удаление или вставка фрагмента доступны из меню, которое появляется  при долгом нажатии на строке любого фрагмента

2.1.3  Путём фиксации длительности циклов движений с помощью секундомера с отсечками

Для некоторых упражнений (например, приседаний) это можно сделать самостоятельно, для большинства других придётся прибегнуть к посторонней помощи.

Кроме того, можно записать раскладку своего друга, соперника и т.д. и попробовать повторить или превзойти их результат!


3 Экран «Секундомер». Использование секундомера.

3.1 Использование секундомера для создания раскладки по темпу

Секундомер с отсечками позволяет делать отсечки времени  в ходе выполнения какого либо упражнения. Если делать отсечки в каждом цикле циклического упражнения, можно получить достаточно подробную и информативную раскладку по темпу, каждый фрагмент которой включает время между соседними циклами движений  и количество повторений равное 1.

После нажатия на кнопку «Стоп» появляется диалог сохранения раскладки секундомера, в котором  задаётся имя раскладки и тип сохранения.

При  выборе пункта «Сохранить и показать» вызывается экран «График», а отсечки отображаются  на графике, что позволяет визуально оценить степень изменения темпа в процессе выполнения упражнения.

При нажатии на стрелку справа  в панели инструментов раскладка загружается в темполидер, где может быть воспроизведена или отредактирована

3.2 Использование секундомера для  записи интервалов времени.

Кроме создания раскладок секундомер может использоваться и в обычном режиме записи отсечек по кругам, например при беге.

4 Экран «Темполидер». Использование темполидера.  

При выборе пункта Главного меню «Темполидер» вы попадаете в основной экран приложения, который позволяет выполнять циклическое упражнение в соответствии с заданной раскладкой по темпу.

Имя выбранной раскладки показано  в верхней части экрана.

Задержку старта для подготовки к выполнению упражнения можно задать при нажатии на кнопку «Задержка, сек».

Если в Настройках включен звук, выполнение упражнения сопровождается звуковыми сигналами, в противном случае можно ориентироваться на цветовые подсказки.

После окончания упражнения включается таймер отдыха, повторное нажатие на кнопку «Старт» запускает следующий подход с повторной задержкой старта.

Список раскладок,  режим редактирования раскладок  и настройки   доступны при нажатии на иконки в  панели инструментов, кроме того в режим редактирования можно перейти при долгом нажатии на любом  фрагменте подхода.

4.1 Экран «Изменить». Редактирование раскладок.

Доступно изменение параметров (время цикла или количество повторений) для всех фрагментов сразу или для любого выбранного фрагмента в зависимости от установки пункта «Выполнить для всех строк».

Изменение, удаление или вставка фрагмента доступны из меню, которое появляется  при долгом нажатии на строке любого фрагмента.

Добавление фрагмента подхода в конец списка фрагментов, а также изменение темпа выполнения упражнения в целом доступно из панели инструментов

При каком либо изменении параметров нагрузки на панели инструментов появляется иконка «Сохранить как», взывающая соответствующий диалог сохранения раскладки под новым именем.

5 Экран «Раскладки». Списки раскладок.  

Экран включает в себя три вкладки, содержащие списки раскладок, сохранённые в базе данных приложения.

Вкладка «Сек» изначально содержит раскладки по темпу, созданные из «Секундомера»

Вкладка «Темп» изначально  содержит раскладки по темпу, созданные или отредактированные и сохранённые из «Темполидера»

Вкладка «Избр» изначально  содержит раскладки по темпу, созданные из главного меню при нажатии на круглую кнопку «+».

Перемещение по вкладкам, удаление или изменение имён раскладок доступно из выпадающего меню при долгом нажатии на строку  с названием раскладки.

Исключение составляют системные раскладки «автосохранение_секундомера» и «автосохранение_темполидера», манипуляции с которыми запрещены на программном уровне.
